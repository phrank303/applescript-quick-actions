# macOS AppleScript & Shell Quick Actions

A collection of macOS automation scripts for Finder, Capture One, 
and video processing – built to keep large photo and video archives 
organized and workflows fast.

## Contents

### 📁 Finder & Folder Icons
Custom folder icon scripts to make macOS more visual and organized.

### 📷 Capture One
Automation scripts for Capture One workflows.

### 🗂️ File Sorting
Finder Quick Actions to sort and rename files automatically.

### 🎬 Video Processing
FFmpeg-based shell scripts for batch video conversion (H.265, 
hardware acceleration on Apple Silicon).

## Requirements
- macOS 12 or later
- Automator (for Quick Actions)
- FFmpeg (for video scripts): `brew install ffmpeg`

## Installation
Each folder contains instructions for its scripts.
